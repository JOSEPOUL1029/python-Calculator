#Python Calculator CLI 

A simple, clean command-Line Calculator built in Python. Built with importable functions and a choice-based menu.

# FEATURES 
- Add, Subtract, Multiply, Divide
-Input validation numbers
-Division by zero protection
-Usable as a standalone CLI app or imported as module
